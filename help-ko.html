📊 XRP 인사이트 대시보드

종합 XRP 시장 분석 플랫폼

🎯 개요

XRP 인사이트 대시보드는 XRP ETF 보유량, 거래소 잔액, 시장 심리 등을 추적하는 무료 실시간 분석 플랫폼입니다. 모든 데이터는 공식 API에서 가져오며 정기적으로 업데이트됩니다.

📈 실시간 데이터

몇 분마다 갱신되는 실시간 가격, ETF 보유량, 거래소 잔액 정보.

🌐 다국어 지원

영어와 한국어 완벽 지원.

📱 모바일 최적화

데스크톱, 태블릿, 모바일에서 작동하는 반응형 디자인.

🤖 AI 기반

Claude AI를 사용한 AI 생성 시장 인사이트 및 분석.

📑 대시보드 탭

📈 거래 탭

GXRP (Grayscale), XRP (Bitwise), XRPC (Canary) 등 미국 상장 XRP ETF의 일일 거래량, 가격, 성과 지표를 포함한 XRP 현물 ETF 거래 데이터를 확인하세요.

💎 보유량 탭

모든 현물 ETF의 총 XRP 보유량을 추적합니다. 어떤 ETF 제공업체가 가장 많은 XRP를 보유하고 있는지 확인하고 대화형 차트로 일일 유입/유출을 모니터링하세요.

🏦 거래소 탭

40개 이상의 주요 암호화폐 거래소의 XRP 잔액을 모니터링합니다. 현재 잔액을 과거 데이터와 비교하여 축적 또는 분배 패턴을 식별하세요.

  • 실시간 데이터 가져오기 - XRPL에서 실시간 잔액을 가져옵니다
  • 과거 비교 - 2025년 2월 24일 기준선과 비교
  • 순 흐름 분석 - 어떤 거래소가 XRP를 얻거나 잃고 있는지 확인

📊 분석 탭

차트, 순위, 흐름 분석으로 거래소 분석을 심층적으로 살펴보세요.

  • 거래소 흐름 분석 신규 - 유입 vs 유출의 시각적 분석
  • 상위 순위 - 보유량별 상위 거래소 확인
  • 데이터 내보내기 - CSV 또는 JSON으로 다운로드

🏆 상위 10K 탭

XRP 부자 목록 - XRPL의 상위 10,000개 최대 XRP 지갑을 탐색하세요.

  • 고래 알림 신규 - 실시간 대규모 거래 알림
  • 크기별 필터 - 상위 25, 50, 100, 500 지갑 보기
  • 지갑 분석 - 잔액 분포 및 지배력 확인

📰 뉴스 탭

여러 소스에서 최신 XRP 및 암호화폐 뉴스를 확인하세요.

  • AI 뉴스 요약 신규 - AI 생성 일일 요약
  • 심리 분석 - 강세/약세 등급 확인
  • 다양한 소스 - CoinDesk, Cointelegraph 등의 뉴스

🤖 AI 인사이트 탭

AI 기반 시장 분석 및 포트폴리오 도구.

  • 기술적 분석 신규 - RSI, MACD, 볼린저 밴드
  • 포트폴리오 트래커 - 보유량 가치 및 손익 계산
  • XRPL 실시간 데이터 - 실시간 원장 통계
  • 가격 시나리오 - 다양한 가격에서의 잠재적 포트폴리오 가치 확인

𝕏 소셜 탭

XRP 커뮤니티 심리 및 소셜 지표를 추적합니다.

  • 심리 점수 - 커뮤니티 심리 게이지 (0-100)
  • Twitter/Reddit 통계 - 팔로워 수 및 활동
  • 트렌딩 토픽 - 인기 XRP 해시태그

✨ 새로운 기능

🐋 고래 알림

대규모 XRP 거래(100만 XRP 이상)에 대한 실시간 알림. 입금, 출금 또는 거래 규모별로 필터링합니다. 거래 세부 정보를 위해 XRPScan에 직접 연결됩니다.

📊 기술적 분석

30일 가격 기록에서 계산된 전문가급 기술 지표:

  • RSI (14일) - 과매수/과매도 신호가 있는 상대강도지수
  • MACD (12, 26, 9) - 교차 알림이 있는 이동평균수렴확산
  • 볼린저 밴드 (20, 2) - 위치 표시기가 있는 가격 변동성 밴드

💰 거래소 흐름 분석

거래소 전반에 걸친 XRP 흐름의 시각적 분석. 순 유출은 일반적으로 강세 심리(매도 압력 감소)를 나타내고, 순 유입은 약세 심리를 나타낼 수 있습니다.

🤖 AI 뉴스 요약기

심리 분석이 포함된 AI 기반 주요 XRP 뉴스 헤드라인 일일 요약. "생성" 버튼을 클릭하여 새로운 요약을 만드세요.

📡 데이터 소스

  • ETF 데이터 - Yahoo Finance API
  • 거래소 잔액 - rippled 노드를 통한 XRP 원장 (XRPL)
  • 부자 목록 - XRPScan API
  • 가격 데이터 - CoinGecko API
  • 뉴스 - CryptoPanic API
  • AI 분석 - Anthropic Claude API

❓ 자주 묻는 질문

데이터가 로드되지 않는 이유는?

백엔드 서버가 깨어나는 중일 수 있습니다 (무료 티어는 콜드 스타트가 있음). 30-60초 기다린 후 새로고침하세요. 문제가 지속되면 인터넷 연결을 확인하세요.

데이터는 얼마나 자주 업데이트되나요?

ETF 데이터: 5분마다. 거래소 데이터: "실시간 데이터 가져오기"를 클릭할 때. 뉴스: 5분마다.

이것은 재정 조언인가요?

아니요. 이 대시보드는 정보 및 교육 목적으로만 사용됩니다. 투자 결정을 내리기 전에 항상 재정 고문과 상담하세요.
